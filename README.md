## (3) 시각화 결과

### 차트1. 소득그룹별 이중배제율

**SQL**
```sql
SELECT 소득그룹,
       COUNT(*) AS 전체인원,
       SUM(이중배제) AS 이중배제인원,
       ROUND(AVG(이중배제) * 100, 1) AS 이중배제율
FROM mz_isolation
GROUP BY 소득그룹
ORDER BY CASE 소득그룹
    WHEN '저소득' THEN 1
    WHEN '중간'   THEN 2
    WHEN '고소득' THEN 3
END
```

**인사이트**  
저소득층의 이중배제율(16.9%)은 고소득층(4.3%)보다 약 4배 높다.
저소득 MZ 6명 중 1명은 음주도, 단체참여도 둘 다 없으며,
분석2의 '고립으로 이어질 수 있음'이 수치로 실증된다. (χ²=40.9, p<0.001)

---

### 차트2. 소득분위별 이중배제율 추이

**SQL**
```sql
SELECT 소득코드,
       소득레이블,
       COUNT(*) AS 인원수,
       SUM(이중배제) AS 이중배제인원,
       ROUND(AVG(이중배제) * 100, 1) AS 이중배제율
FROM mz_isolation
WHERE 소득코드 <= 7
GROUP BY 소득코드, 소득레이블
ORDER BY 소득코드 ASC
```

**인사이트**  
전반적으로 소득이 낮아질수록 이중배제율이 상승하는 패턴이 나타난다.
소득없음 구간(19.4%)에서 고립이 가장 높고, 400-500만 구간(2.1%)에서 가장 낮다.
단, 500-600만 구간(n=57)은 표본이 작아 수치가 불안정하므로 해석에 주의가 필요하다.

## (4) 핵심 발견 요약

1. **이중배제율 격차 (χ²=40.9, p<0.001):** 저소득 MZ의 16.9%는 음주도 없고 단체참여도 없음 — 고소득(4.3%)의 약 4배
2. **소득분위별 패턴:** 소득이 낮아질수록 이중배제율 상승. 소득없음 구간 19.4%로 최고

**결론:** 저소득층에게 음주의 자리를 대체한 것은 '건강한 교류'가 아니라 '고립'이었다.
분석2의 가설 '금주 트렌드가 저소득층에게는 고립으로 이어질 수 있음'이 수치로 실증되었다.

**한계:** 본 분석은 횡단면 데이터로 인과관계가 아닌 동반 패턴을 확인한 결과다.
이중배제의 원인이 저소득인지, 학력·고용상태 등 다른 요인의 영향인지는 추가 분석이 필요하다.
